# TaskCLI

## Problema
Estudantes têm dificuldade em organizar tarefas do dia a dia.

## Solução
Aplicação CLI para gerenciar tarefas.

## Público
Estudantes

## Funcionalidades
- Adicionar tarefa
- Listar tarefas
- Remover tarefa

## Executar
python src/app.py

## Testes
pytest

## Lint
ruff check .
